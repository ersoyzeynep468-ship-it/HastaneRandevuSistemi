# HastaneRandevuSistemi
Bu proje, C# ve Windows Forms kullanılarak geliştirilmiş bir hastane randevu sistemidir. Kullanıcıların branş, doktor, tarih ve saat bilgilerini alabilmelerini sağlar.

Özellikler
Kullanıcı giriş
Branş Seçimi
Doktor Seçimi
Tarih ve saat seçimi
Randevu onay ekranı
Hatalı kızlar için uyarı mesajları
Kullanılan Teknolojiler
C#
Windows Formları
.NET Çerçevesi

# HastaneRandevuSistemi
Bu proje, C# ve Windows Forms kullanılarak geliştirilmiş bir hastane randevu sistemidir. Kullanıcıların branş, doktor, tarih ve saat bilgilerini alabilmelerini sağlar.

ÖZELLİKLER

-Kullanıcı giriş

-Branş Seçimi

-Doktor Seçimi

-Tarih ve saat seçimi

-Randevu onay ekranı

-Hatalı kodlar için uyarı mesajları

Kullanılan Teknolojiler

C#

Windows Formları

.NET Çerçevesi

Projenin yönetimi

Bu depoyu indirin (Code > HTTPS yi kopyalayın).

Visual Studio ile depoyu klonlaya basın, HTTPS yi yapıştırın sonra kopyalayın.

Süprizler

Ekstra bir veya dosya paketi gerektirmemektedir.

.NET Framework yüklü olması yeterlidir.
